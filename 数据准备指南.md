# YOLOv8 数据准备指南

由于网络问题，提供多种数据获取方案。

## 方案一：使用Ultralytics自动下载（推荐）

```python
# 最简单的方式 - Ultralytics会自动下载COCO128
from ultralytics import YOLO

# 创建模型
model = YOLO('yolov8n.pt')

# 使用coco128.yaml会自动下载数据集
results = model.train(data='coco128.yaml', epochs=10, imgsz=320)
```

## 方案二：手动下载数据集

### 1. 使用浏览器下载

**COCO128数据集（~20MB）：**
- 官方地址：https://github.com/ultralytics/assets/releases/download/v0.0.0/coco128.zip
- 国内镜像：https://ghproxy.com/https://github.com/ultralytics/assets/releases/download/v0.0.0/coco128.zip

下载后解压到 `yolo_project/data/coco128/` 目录。

### 2. 使用迅雷/IDM等下载工具

如果浏览器下载慢，使用下载工具：
1. 复制上面的链接
2. 打开迅雷/IDM
3. 新建下载任务，粘贴链接
4. 下载完成后解压到 `data/coco128/` 目录

### 3. 百度网盘下载

如果GitHub无法访问，可以使用百度网盘：
- 搜索 "COCO128 dataset yolo"
- 通常有热心网友分享

## 方案三：使用自己的图片

### 快速开始（无需下载）

你可以使用桌面上的任意图片进行测试：

```python
# detect.py 修改示例
# 直接检测桌面上的图片

from ultralytics import YOLO
import cv2

# 加载预训练模型（会自动下载）
model = YOLO('yolov8n.pt')

# 使用桌面上的图片
img_path = r'C:\Users\Yyt\Desktop\你的图片.jpg'  # 修改为你的图片路径

# 进行检测
results = model(img_path)

# 显示结果
for result in results:
    result.show()
```

## 方案四：创建示例数据集

如果以上方法都不可行，可以创建合成数据：

```python
# 使用OpenCV创建示例数据
import cv2
import numpy as np
from pathlib import Path

# 创建目录
data_dir = Path('data/mydata')
for split in ['train', 'val']:
    (data_dir / 'images' / split).mkdir(parents=True, exist_ok=True)
    (data_dir / 'labels' / split).mkdir(parents=True, exist_ok=True)

# 生成10张示例图片
for i in range(10):
    # 创建图片
    img = np.random.randint(0, 255, (480, 640, 3), dtype=np.uint8)
    
    # 添加一些形状
    cv2.rectangle(img, (100, 100), (200, 200), (255, 0, 0), -1)
    cv2.circle(img, (400, 300), 50, (0, 255, 0), -1)
    
    # 保存
    cv2.imwrite(str(data_dir / 'images' / 'train' / f'img_{i}.jpg'), img)
    
    # 创建标注文件
    with open(data_dir / 'labels' / 'train' / f'img_{i}.txt', 'w') as f:
        f.write('0 0.234 0.312 0.156 0.208\\n')  # 矩形
        f.write('1 0.625 0.625 0.078 0.104\\n')   # 圆形

print("示例数据集创建完成！")
```

## 方案五：使用Python代码直接下载

创建一个新文件 `download_data.py`：

```python
import urllib.request
import zipfile
from pathlib import Path

def download_with_progress(url, dest):
    """下载文件并显示进度"""
    def report(block_num, block_size, total_size):
        downloaded = block_num * block_size
        percent = min(downloaded * 100 / total_size, 100)
        print(f'\\r下载进度: {percent:.1f}%', end='')
    
    print(f'开始下载: {url}')
    urllib.request.urlretrieve(url, dest, reporthook=report)
    print('\\n下载完成!')

# 下载COCO128
url = 'https://github.com/ultralytics/assets/releases/download/v0.0.0/coco128.zip'
dest = 'coco128.zip'

try:
    download_with_progress(url, dest)
    
    # 解压
    print('解压中...')
    with zipfile.ZipFile(dest, 'r') as zip_ref:
        zip_ref.extractall('data/')
    
    print('完成!')
    
except Exception as e:
    print(f'错误: {e}')
    print('尝试使用镜像源...')
    # 如果失败，使用镜像
    url = 'https://ghproxy.com/' + url
    download_with_progress(url, dest)
```

## 常见问题

### Q: 下载速度太慢怎么办？
**A:** 
- 使用迅雷等下载工具
- 尝试国内镜像站
- 使用手机热点下载
- 请朋友帮忙下载后传给你

### Q: 下载后提示文件损坏？
**A:**
- 重新下载
- 检查文件大小是否正确（coco128.zip约20MB）
- 使用7-zip等工具修复

### Q: 不想下载数据集，能直接测试吗？
**A:** 可以！
```python
from ultralytics import YOLO
import numpy as np

# 加载模型
model = YOLO('yolov8n.pt')

# 创建随机测试图片
img = np.random.randint(0, 255, (640, 640, 3), dtype=np.uint8)

# 检测
results = model(img)
results[0].show()
```

### Q: 可以使用自己的数据集吗？
**A:** 可以！准备数据格式：
```
mydata/
├── images/
│   ├── train/     # 训练图片
│   └── val/       # 验证图片
└── labels/
    ├── train/     # 训练标注（YOLO格式）
    └── val/       # 验证标注
```

YOLO标注格式（每行一个目标）：
```
<class_id> <x_center> <y_center> <width> <height>
```

示例：
```
0 0.5 0.5 0.3 0.4
1 0.2 0.3 0.1 0.2
```
